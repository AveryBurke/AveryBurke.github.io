Pie

Configurable data visualization app build with React, D3.js and TypeScript. Visualize users in an imaginary database. Use the side bar components to select a column for slices (as in a pie chart) and rings. Shuffle the slices and rings by dragging the corresponding elements on the side bar. And filter out users with the filter.
The visualization is generated using D3 rendered to a canvas element. The placement of the data points is calculated using Lloyd’s algorithm for generating a Voronoi mesh. The calculations themselves are run on the client’s GPU using WebGPU compute. The state management uses the React Context API. The different layers of the visualization are run from inside web workers and rendered using offscreencanvas. Finally, the sidebar components are reusable. View the code.

G(PU)ame Of Life

The game of life. Implemented using WebGPU. Adapted from this codelab tutorial. The code is warped in a few React components. I’ve added user interactions. The user can click on a cell to change its state, or use the sliders to resize the board and change the generation speed. view the code on github. Check if your browser has WebGPU
